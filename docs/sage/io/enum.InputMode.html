Expand description

The different types of input modes a program might use.

Variants§

§

StdinChar

Standard input modes (The standard interface is typically used for command line programs, and for all pipelines between programs) Standard input (ASCII character)

§

StdinInt

Standard input (integer)

§

StdinFloat

Standard input (float)

§

Random

Special input modes A random number

§

DPad(Direction)

User input modes (These should typically be used for games or other interactive programs) Input from a D-Pad

§

Button

Input from a button (0=not pressed, 1=pressed)

§

Keyboard

Input from keyboard (ASCII character)

§

JoyStick(Direction)

Input from a JoyStick the degree of displacement in a given direction (from -128 to 128).

§

Clock

Physical sensor input modes (These should typically be implemented for devices that measure the physical world, like a smart watch or a smart phone) Time (in seconds) since the program started

§

Accelerometer(Axis)

Input from an accelerometer (in meters per second per second) in a given axis

§

Gyroscope(Axis)

Input from a gyroscope (in degrees per second) around a given axis

§

Microphone

Input from a microphone (frequency in hertz)

§

RedLight

Environment sensor input modes (These should typically be implemented for devices that measure the environment, like a weather station or a smart thermostat) Red light intensity (in lux)

GreenLight

Green light intensity (in lux)

BlueLight
